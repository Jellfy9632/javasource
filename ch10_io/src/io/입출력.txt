입출력(Input/Output)

-기본입력 : 키보드
-기본출력 : 화면

- System.out (화면)
- System.in (키보드)
ㄴnew Scanner(Systme.in) : 키보드에서 입력받는 새로운 스캐너

-입력장치(키보드,파일,객체..)에서 읽어오기

1) text 파일 : 문자 스트림
2) text + 모든파일 : 바이트스트림

- 스트림(stream): 서로 연결된 두 대상의 연결통로
 - 스트림은 양방향 사용 불가
 - 입력, 출력을 따로 처리



-File : 자바에서 파일(텍스트,이미지,동영상..) 처리를 담당하는 객체
 -윈도우에서 사용하는 경로 구분자 : ;(세미콜론)
  유닉스(리눅스) :  : (콜론)

 -윈도우에서 사용하는 이름 구분자 : \
 -유닉스(리눅스)에서 사용하는 이름 구분자 : /

 -주요 메소드
  1) 파일 정보 추출
  getName() : 파일명
  getAbsolutePath() : 경로포함 파일명
  getParent() : 부모 폴더
  lastModfied() : 최종수정일자

  2) 파일 / 디렉토리 생성
    mkdir() : 만들고자하는 디렉토리의 상위 디렉토리가 존재하지 않을경우 생성 불가
    mkdirs() : 만들고자하는 디렉토리의 상위 디렉토리가 존재하지 않을경우 상위 디렉토리까지 생성
    createNewFile() : txt 파일생성

    



 File.separator(File.separatorChar) : 운영체제에 맞춰 구분자 생성

  3) 디렉토리 정보추출
 isDirectory() : true 일 때
